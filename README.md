# Charader Projektet

Ett enkelt webapp-projekt byggt med HTML, CSS och JavaScript.

## Installation

Ingen installation behövs – öppna bara `index.html` i din webbläsare.

## Struktur

- `index.html` – startsida
- `style.css` – stilmall
- `script.js` – JavaScript-kod
- `cards.js` - Data
