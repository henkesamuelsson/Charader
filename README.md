# Charader-spelet ⚡ React + Vite

Migrerad från vanilla HTML/JS till React + Vite.

## Kom igång

### 1. Installera beroenden
```bash
npm install
```

### 2. Starta dev-server
```bash
npm run dev
```

### 3. Bygg för produktion
```bash
npm run build
```
Bygget hamnar i `dist/` och kan hostas var som helst (Netlify, GitHub Pages, etc.)
